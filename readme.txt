Block AdBlock detects site visitors who have enabled adblocking software, and allows publishers to engage these users via customized display messages.

== Description ==
Installing the Block AdBlock plugin allows publishers to lock their content until users disable their adblock add-ons. Locked content remains SEO friendly.

For advanced analytics and revenue recovery solutions install: [Admiral Adblock Analytics](https://wordpress.org/plugins/admiral-adblock-suite/)

= Block AdBlock Features =
* Random class naming to make it harder for Block AdBlock to be blocked
* The ability to customize your message across 3 different display units/layouts
* Message delay in seconds

== Installation ==
Upload the Block AdBlock plugin to your blog, Activate it, then enable it from the Block AdBlock Settings page.

You're all done.

For advanced analytics and revenue recovery solutions install: [Admiral Adblock Analytics](https://wordpress.org/plugins/admiral-adblock-suite/)
